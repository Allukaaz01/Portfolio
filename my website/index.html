About Me

A 20-year-old Web Developer and OFPPT graduate in Digital Development (WDOFS), I specialize in building modern, high-performance web applications with a strong focus on both functionality and user experience. My technical expertise spans a wide range of technologies, including Python, MySQL, SQL Server, PHP, and JavaScript, along with front-end development using HTML, CSS, and advanced UI/UX design with Figma. I am highly proficient in modern frameworks and libraries such as Laravel and React, and I bring interfaces to life using powerful animation tools like GSAP and Three.js. My skill set allows me to handle full-stack development, from designing intuitive user interfaces to developing robust back-end systems and managing databases efficiently. Driven by curiosity and continuous learning, I am always exploring new technologies and pushing the boundaries of web development to create innovative, scalable, and visually engaging digital experiences.
